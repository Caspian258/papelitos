# Bitácora del Proyecto — Papelitos Digital

> Este archivo documenta el progreso real del proyecto, fecha por fecha. Su propósito es evitar repetir errores: cada vez que algo salga mal, se anota qué pasó y cómo se resolvió, para no caer en lo mismo después.

**Reglas para Claude Code:**
- Nunca agregues una entrada aquí por iniciativa propia.
- Solo se actualiza cuando el usuario lo pide explícitamente.
- Cuando se te indique actualizar, ejecuta `date` en la terminal para obtener la fecha y hora real — nunca la inventes ni la asumas.
- Antes de empezar una tarea nueva, revisa este archivo completo para verificar si un error similar ya ocurrió.
